# benchkit

## Development setup

From the root of the repository, open a terminal and run:

```bash
make ready
```

The environment is ready if the command passes.

## Working locations

* `notebooks/` for demos
* `src/benchkit/` for the package implementation
* `tests/` for tests
* `data/` for provided dummy test data

## Useful commands

```bash
make install
make test
make lint
make format
make clean
```
